# AlchemyForge — Independent Developer Services

A modern, premium service website for a single developer offering interactive web systems, problem-solving, and custom digital solutions with a sophisticated, professional user experience.

## Overview

AlchemyForge is a service-focused website built to present an independent developer's offerings to potential clients. It showcases service areas, development process, technical expertise, and featured case studies—positioning a solo developer as a capable, professional service provider. The design emphasizes quality, intention, and craftsmanship in building custom digital solutions.

## Features

- **Premium Hero Section** - Animated background with gradient effects and smooth reveal animations
- **Service Offerings** - Two core service areas with clear benefits and outcomes
- **Featured Case Studies** - 5 completed projects demonstrating real-world work and results
- **Development Process** - Transparent methodology showing how projects are delivered (4 phases)
- **Technical Expertise** - 20+ technologies and tools mastered and available for projects
- **About Section** - Personal story and key statistics (problems solved, projects built)
- **Contact Form** - Functional message submission with visual feedback
- **Custom Cursor** - Unique cursor design that responds to interactive elements
- **Smooth Animations** - Scroll reveals, fade-in effects, gradient shifts, and floating elements
- **Dark Theme** - Premium dark mode with green and cyan accents
- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Mobile Navigation** - Hamburger menu with smooth interactions on smaller screens

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Styling**: Tailwind CSS (utility-first), Custom CSS animations
- **SDKs**: Element SDK (configuration), Data SDK (data management)
- **Features**: Intersection Observer (scroll reveals), LocalStorage, Smooth scroll
- **Design Pattern**: Component-based vanilla JavaScript rendering
