ABOUT

Independent Builder

I'm Sharek Jabir, a student developer from Dhaka who builds under the name GlitchForge. This isn't about copying trends or stacking frameworks—it's about understanding how systems think, respond, and feel.

I focus on the intersection of logic and design. My work explores how code can create expressive interfaces, how small interactions shape user experience, and how clean structure supports everything else. Every project is a chance to solve real problems while learning something new.

I don't pretend to be a senior developer. I'm building foundations—practicing algorithms, studying architecture, and creating projects that feel intentional rather than assembled. The goal isn't just output; it's depth, clarity, and craft.
